# Giants-Din-Game

My project is called Giants Din. It is a game where the player tries to get the gold and leave without getting caught by the dragon. I made my project with snoozefest and this was my first time making a game. Snoozefest seemed like the perfect opportunity to commit and try out game development. I made my game with claude AI. it gave me the instructions to learn and execute my idea. There was a lot of trouble shooting during this project. At first I wanted it the to be a giant but couldn't find a picture of it sleeping, so I had to adapt. I also was having trouble getting the dragon to notice if the person was in its vicinity. I eventually figured it out but this was crucial because the end game screen wouldn't display unless it noticed that the character touched the dragon. Through this project I learned about itch.io and Godot.
